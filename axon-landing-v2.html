Questions

Frequently
Asked.

Does it impact performance?
A

Axon

Only what it ships. Everything is bundled, minified, and tree-shaken — no overhead beyond the code you actually use.
Where does my code get hosted?
A

Axon

Files go to Cloudflare R2, served from cdn.axon.dev — dedicated CDN with global edge coverage and zero cold starts.
What if I deploy something wrong?
A

Axon

Hit rollback. Previous version is live in seconds. Fix locally, redeploy when ready — no drama, no downtime.
How do agents stay in sync?
A

Axon

Shared project context via MCP. Each agent sees the same file tree, types, and build state — no conflicts, no drift, no duplicate work.
Which AI models are available?
A

Axon

GPT-4o, Claude Sonnet, Gemini Pro. The orchestrator picks automatically, or you can pin any model per task from a toggle.
